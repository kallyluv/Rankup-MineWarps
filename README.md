# About
Rankup-MineWarps is a simple prisons plugins that allows staff to easily manage rankups and minewarps.
```
Commands available to all (rankup.command.default)
- rankup
- rankupmax
- mines
- mine
Commands restricted (rankup.command.admin)
- setlevel
- createminewarp
- deleteminewarp
```
